🔒 Segurança na Internet

Site responsivo desenvolvido para a disciplina de **Interface Web**, com o objetivo de apresentar conceitos e boas práticas de segurança na internet.

📖 Sobre o projeto

O site aborda temas relacionados à segurança digital, como:

- Proteção de dados pessoais;
- Senhas seguras;
- Phishing;
- Antivírus;
- Atualizações de sistemas;
- Boas práticas para navegação segura.

Além do conteúdo informativo, o projeto conta com:

- Formulário de contato;
- Galeria de imagens;
- Painel de fotos desenvolvido com CSS Grid;
- Layout responsivo para diferentes tamanhos de tela.

 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- CSS Grid
- Flexbox
- Google Fonts
- Modern Normalize

📱 Responsividade

O site foi desenvolvido para funcionar corretamente em dispositivos móveis, tablets e desktops, adaptando o layout entre aproximadamente 375px e 1920px de largura.
